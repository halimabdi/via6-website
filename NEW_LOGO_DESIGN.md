## Usage Guidelines

### DO ‚úÖ
- Use on solid backgrounds (dark or light)
- Maintain aspect ratio when resizing
- Use logo.svg for primary applications
- Use logo-white.svg on dark/photo backgrounds
- Use logo-icon.svg for small/square spaces
- Ensure minimum height of 32px for readability

### DON'T ‚ùå
- Stretch or distort the logo
- Change gradient colors
- Separate icon from text in main logo
- Add effects (drop shadow, glow, etc.)
- Place on busy/patterned backgrounds
- Use at less than 32px height

### Minimum Sizes
- **Full logo**: 32px height minimum
- **Icon only**: 24px √ó 24px minimum
- **Print**: 0.5 inches / 1.3cm minimum

### Clear Space
- Maintain minimum spacing around logo
- Clear space = height of "AI" badge (14px)
- No text or graphics within clear space
